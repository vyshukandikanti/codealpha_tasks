# CodeAlpha Data Analytics Internship — Power BI Dashboards

All 4 dashboards I built during my **Data Analytics Internship at CodeAlpha**, created in **Microsoft Power BI**.

## 📊 Dashboards

### 1. Financial Health Dashboard
Revenue & profit trends, budget vs actual, balance sheet (Assets / Liabilities / Equity), and a cash-flow waterfall — 3 interactive pages.
`CodeAlpha_FinancialHealthDashboard.pbix`

### 2. HR Analytics Dashboard
Workforce overview, attrition analysis (key finding: **overtime + low job satisfaction drive higher attrition**), and recruitment performance.
`CodeAlpha_HRAnalytics.pbix`

### 3. Real Estate Market Dashboard
A geographic **map of 2,000 listings across India**, pricing & rental-yield analysis, and market trends (price index, demand/supply, home-loan rates).
`CodeAlpha_RealEstate.pbix`

### 4. Education Performance Dashboard
Student performance, the **real drivers of scores** (tutoring, internet access, parent education), and school resource allocation.
`CodeAlpha_EducationPerformance.pbix`

## 🛠️ Tools & Skills
- Microsoft Power BI — multi-page interactive reports
- DAX measures (KPIs, rates, averages)
- Data modeling & preparation
- Map visuals, slicers, custom themes & dashboard design

## 📌 How to View
Download any `.pbix` file and open it in **Power BI Desktop** (free).
